## Tabs
Tabs allow users to toggle between different sections of content. Although its similar to expaders, tabs have a restriction to the
number of tabs and much more limitations of content within the headings. Consider using other patterns more than 3 tabs.  


<div class="formsflow-tab">
  <button>Tab label</button>
  <button>Tab label</button>
  <button>Tab label</button>
</div>


  ```html dark
    <div class="formsflow-tab">
        <button>Tab label</button>
        <button>Tab label</button>
        <button>Tab label</button>
    </div>
  ```

#### Active state
There is no need to add class to buttons since they are using psuedo-class. To force the appearance of active state use `.active` class

<div class="formsflow-tab">
  <button class="active">Tab label</button>
</div>

```html dark
<div class="formsflow-tab">
  <button class="active">Tab label</button>
</div>
```
